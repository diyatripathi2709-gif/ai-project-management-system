# AI Project Management System

## Overview
This project demonstrates an AI-powered project management workflow for an imaginary company.

## Features
- Project Charter
- Risk Register
- Issue Log
- Stakeholder Register
- Dashboard Tracking
- AI-generated project summaries

## Tools Used
- ChatGPT
- Google Sheets
- Canva
- Excel
- Power BI

## Objective
To automate project tracking and reporting using AI and no-code tools.
